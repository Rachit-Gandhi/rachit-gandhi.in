# Gator — CLI RSS Feed Aggregator

- Built a CLI RSS aggregator supporting user registration, feed management, and browsing with PostgreSQL backend.
- Implemented schema migrations using **goose** and type-safe query generation via **sqlc**.
- Added concurrency-aware feed fetching with rate-limit handling.
- Distributed as a statically compiled binary using `go build` and `go install`.
