# scrum-cli — Terminal Scrum Workflow Assistant

- Built a command-line tool to manage day-to-day Scrum workflows directly from the terminal.
- Designed flows for sprint standups, progress snapshots, and lightweight task updates.
- Focused on speed and low-friction usage so updates can be done without leaving coding context.
- Shipped as a simple CLI package suitable for local developer workflows.
